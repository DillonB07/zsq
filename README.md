# zsq
> a complex language with high level programming and moderate syntax. 

## About
### Usage
1. Clone this repository: 
```
$ git clone https://github.com/JBYT27/zsq.git
```

2. Run zsq

Create a file named `main.zsq`, which will contain all of your code.
```
zsq main.zsq 
```

And now you're done!
